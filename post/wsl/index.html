位置迁移#

先导出再导入即可实现迁移。

1 关闭系统#

确认系统运行情况：

wsl --list --verbose
# 或
wsl -l -v

关闭系统：

wsl --shutdown

2 迁移#

导出到事先创建好的D:\Kali目录：

# wsl --export <系统名> <导出文件路径>
wsl --export kali-linux D:\Kali\Kali.tar

导入：

# wsl --import <系统名> <安装位置> <导入文件位置> <WSL版本号>
wsl --import Kali D:\Kali D:\Kali\Kali.tar --version 2

删除迁移前的系统：

wsl -l -v
# 结果如下
  NAME          STATE           VERSION
* kali-linux    Stopped         2
  Kali          Stopped         2
  
# 删除第一个，即迁移前的系统
wsl --unregister kali-linux

完成迁移。

Kali安装桌面环境#

apt update && apt upgrade -y

apt install -y kali-win-kex

apt install -y zsh zsh-syntax-highlighting zsh-autosuggestions

kex参数：

# 窗口模式
kex --win -s
# 增强会话模式
kex --esm --ip -s
# 无缝模式
kex --sl -s
